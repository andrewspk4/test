Dependencies: zeromq, sqlite3, readline-sync
Install dependencies:

1) npm install zeromq
2) npm install sqlite3
3) npm install readline-sync

Run project:

1) Run "create_db.js" 
Creates DB and table, insert users information in table.

2) Run "pubber.js".
Accepts 2 command line options - pub and sub ports.

3) Run "subber.js".
Accepts 2 command line options - pub and sub ports. (Reverse port order relative to "pubber.js")
